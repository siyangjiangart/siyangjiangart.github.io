Dolce Gabbana 3D Web Design

3D WEB

This is a 3D website inspired by the Dolce & Gabbana Blue Ceramic Sofa. Users can interact with the website, rotate the model, and view the details of the chair. This represents a highly promising future shopping method, where users can directly inspect product details on a shopping website before making a purchase. The 3D model can also be used in mall advertisements, large interactive projections, or display screens. If you're interested, you can check out the original 3D website: https://siyangjiangart.github.io/dolcegabbana3D/. Thank you!
